Структура
---

```
cup/
   - base.obj
   - transforms.json
   - data
      - cup
        -image1.jpg
        -image2.jpg
        -image3.jpg
  ...
```

Файл `transforms.json` содержит данные по для тренировки нейросети.
Каталог `data` содержит подкатолог с одноименным названием объекта, содержащий изображение самого объекта в реальном мире.
